# HabitFlow

Personal Habit Tracker API

## Features
- User registration and login with JWT authentication
- Full CRUD for habits — create, read, update, delete
- Daily log entries per habit with date tracking
- Streak calculation — current streak and longest streak
- Completion rate analytics per habit
- Daily email reminders via cron job at 8 AM
- Request validation with Zod on all endpoints
- Swagger UI at /api-docs with full endpoint documentation
- Error handling middleware — no unhandled crashes
- Deployed live on Render with Supabase Postgres
