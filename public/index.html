Features

What's under the hood

A focused backend — no bloat, just the things that matter for habit tracking.

Cookie-based JWT login
Login sets an HttpOnly cookie, and protected routes read that JWT through the authentication middleware.
Daily log endpoints
Create and fetch habit logs through /logs/:id, with a same-day duplicate check before a new log is saved.
Streak analytics
Current streak, longest streak, total completions, and completion rate, computed per habit at /analytics/:id.
Zod validation
Request bodies and route params are validated with Zod before controller logic writes to the database.
OpenAPI 3.0.0 docs
Swagger UI is mounted at /api-docs from docs/swagger.yaml.
Health checks and reminders
A simple GET /health returns server status and timestamp, while node-cron runs daily reminder emails through Nodemailer.
